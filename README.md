# Kingdom Of God Center — Table Guide

A premium, modern web application for managing event tables, guests, and seating arrangements. Built for speed, reliability, and ease of use.

## Features
- **Real-time Synchronization**: Uses Firebase to keep table data instantly in sync across all devices.
- **Premium UI**: Modern dark theme with glassmorphism, smooth animations, and gold accents.
- **Interactive Floor Map**: Visual representation of the venue to easily locate tables and guests.
- **Waiter Assistant**: Instant search to help waiters escort guests to their assigned tables.
- **Table Management**: Quickly assign or remove guests from tables.

## How to Host on GitHub Pages

Since this project uses plain HTML, CSS, and JavaScript, hosting it on GitHub Pages is incredibly simple.

1. **Create a GitHub Repository**: 
   - Go to [github.com](https://github.com) and create a new repository.
2. **Upload the Files**: 
   - Upload `index.html`, the `css` folder, and the `js` folder to your new repository.
3. **Enable GitHub Pages**:
   - Go to your repository's **Settings**.
   - On the left sidebar, click on **Pages**.
   - Under the "Build and deployment" section, choose **Deploy from a branch**.
   - Select the `main` (or `master`) branch and the `/ (root)` folder.
   - Click **Save**.
4. **Access Your Site**:
   - In a few minutes, your site will be live! GitHub will show you the link at the top of the Pages settings page.
